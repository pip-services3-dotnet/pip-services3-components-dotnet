Detailed Description

Basic component factory that creates components using registered types and factory functions.

Factory factory = new Factory();
factory.RegisterAsType(
new Descriptor("mygroup", "mycomponent1", "default", "*", "1.0"),
MyComponent1 );
factory.Register(
new Descriptor("mygroup", "mycomponent2", "default", "*", "1.0"),
(locator) => {return new MyComponent2();
});
factory.Create(new Descriptor("mygroup", "mycomponent1", "default", "name1", "1.0"))
factory.Create(new Descriptor("mygroup", "mycomponent2", "default", "name2", "1.0"))

object PipServices3.Components.Build.Factory.CanCreate ( object  locator)

Checks if this factory is able to create component by given locator.

This method searches for all registered components and returns a locator for component it is able to create that matches the given locator.If the factory is not able to create a requested component is returns null.
